Construindo páginas para internet com Bootstrap

O bootstrap é um dos frameworks mais conhecidos e utilizados no desenvolvimento front-end, devido principalmente à sua produtividade. Trata-se de um framework para desenvolvimento de sites responsivos, com uma vasta biblioteca de componentes que facilita e agiliza o desenvolvimento de páginas web (como esta aqui).

Em conjunto com os elementos HTML e estrutura DOM são aplicados vários estilos CSS e até mesmo recursos em JavaScript. No Bootstrap há uma série de classes CSS que podem ser utilizadas facilmente com elementos divs e outros elementos do HTML. Além disso, também pode ser considerado uma coleção de ferramentas que integra componentes do jQuery, assim facilitando o uso das classes CSS do Bootstrap em tags HTML para melhor organizar a estrutura do site, melhorar visualmente os componentes da página, usar recursos de responsividade e JavaScript em conjunto com CSS para criar efeitos elegantes.
